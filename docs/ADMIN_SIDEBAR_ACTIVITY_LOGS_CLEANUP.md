# Admin Sidebar and Activity Logs Cleanup

Updates included:

- Removed the logo, company name, and console label from the admin slide-out sidebar.
- Kept the sidebar close button and navigation links.
- Removed the Back button from the Activity Logs header.
- Cleaned up Activity Log rows so the text no longer feels squeezed on the right side.
- Replaced the large `See change` button with a smaller `View details` link.
- Made the activity title clickable, for example `Manual Ticket Created`.
- Clicking the activity title or `View details` opens a dark themed modal with the full activity details.

No database or ER ticket logic was changed in this update.
