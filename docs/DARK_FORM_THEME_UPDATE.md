# Dark Form Theme Update

This pass removes the bright white form surfaces from the customer request form and internal/manual ticket forms.

Updated styling:

- Customer request form inputs/selects/textareas now use the same dark navy PHP-style field background.
- CSR / Team Leader / CSR Manager manual ticket fields now use dark inputs as well.
- Disabled fields, dropdown options, placeholders, checkboxes, and upload boxes were adjusted to stay readable in the dark theme.
- Focus state uses a cyan glow to match the portal theme.
- Mobile and tablet form fields keep larger touch-friendly height without switching back to white.
