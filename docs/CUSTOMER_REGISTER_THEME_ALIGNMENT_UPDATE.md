# Customer Register Theme + Policy Modal Update

Updated the customer registration page styling to better match the dark portal theme.

Changes included:

- Improved customer account creation form width, spacing, and alignment.
- Fixed input/icon alignment for email, phone, password, and confirm password fields.
- Improved password visibility toggle alignment.
- Added more consistent dark-card styling for the Terms and Privacy agreement area.
- Updated Terms and Conditions modal to match the dark theme instead of the previous white popup.
- Updated Privacy Policy modal to match the dark theme.
- Improved modal header, footer, close button, scroll area, and section headings.
- Kept the existing account creation logic unchanged.

No database or ER ticket logic was changed in this update.
